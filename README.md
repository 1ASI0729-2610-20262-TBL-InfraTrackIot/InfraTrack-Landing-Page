# InfraTrack Landing Page

Landing page institucional para **InfraTrack**, una plataforma de monitoreo IoT orientada al control de maquinaria pesada y flotas en el sector ferretero y construcción.

Este proyecto fue desarrollado como parte de un trabajo universitario para presentar una propuesta moderna, clara y visualmente potente de una solución open source para operaciones logísticas críticas.

## Descripción

La página presenta de manera visual los beneficios de InfraTrack, incluyendo:

- Monitoreo 360° de combustible, GPS y horas de uso.
- Soluciones para dueños de ferreterías y gestores de flota.
- Paneles informativos con métricas operativas y planes escalables.
- Sección de contacto y navegación responsiva.

## Características

- Diseño responsive adaptado a escritorio, tablet y móvil.
- Navegación fija con menú hamburguesa en pantallas pequeñas.
- Secciones dinámicas con animaciones de revelado.
- Tarjetas informativas para tecnología, sectores y planes.
- Footer institucional con información de contacto y suscripción.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3
- Bootstrap Icons
- Google Fonts
